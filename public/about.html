About Secure chat

A secure, scalable real‑time messaging platform for teams and communities. Channels, DMs, presence, media uploads, and a modern API—ready to deploy anywhere.

Real‑time JWT Auth Postgres + Prisma Redis Scale R2/S3 Media Prometheus Metrics

What we built

Real‑time rooms & DMs

Joinable rooms with presence and typing. Direct messages for one‑to‑one chats.

Secure authentication

Hashed passwords, JWT sessions, and per‑socket authorization for WebSockets.

Scales horizontally

Redis adapter shares presence and events across multiple server instances.

Attachments & media

S3‑compatible storage (R2) with signed URLs, size and type validation.

Observability

Structured logs and Prometheus metrics for latency, errors, and throughput.

Modern API

REST endpoints for auth, uploads, history; Socket.IO for bi‑directional events.

FAQs

Is Secure chat open source?
This demo is source‑available for learning and iteration.
How do I deploy it?
Provision Postgres, Redis, and an R2/S3 bucket. Configure `.env`, then deploy the Node service.
What about end‑to‑end encryption?
This version focuses on transport security and server‑side protections. E2EE can be explored as a future enhancement.
